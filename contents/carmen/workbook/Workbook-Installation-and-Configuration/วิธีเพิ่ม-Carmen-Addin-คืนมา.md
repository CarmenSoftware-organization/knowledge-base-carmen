# Tab Addin Carmen หายไป ไม่สามารถใช้งานได้

Sample case : หลังจากติดตั้งเสร็จแล้ว เมื่อกลับมาเปิด excel แต่ Carmen add in หายไป

Cause of problem : Excel มีการ block add in  
Solution : วิธีแก้ไขเพิ่ม add in กลับมา  
![](_images/carmen-addin-a76a87162e/img-001.png)  
1\.Click “File”

![](_images/carmen-addin-a76a87162e/img-002.png)  
  
  
  
  
Click “option”![](_images/carmen-addin-a76a87162e/img-003.png)

2\.Click “Add\-ins”  
3\.Cick GO  
![](_images/carmen-addin-a76a87162e/img-004.png)  
  

  
4\. Check box at “Carmen AddIn”

5\.Click “OK” to complete the process\.  
![](_images/carmen-addin-a76a87162e/img-005.png)  
Excel จะแสดง Carmen Add\-in ขึ้นมาตาม สามารถดำเนินการใช้งานได้ตามปกติ
